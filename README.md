# Hermes WebUI Pro

基于 Vue3 + NaiveUI + FastAPI 的 Hermes Agent Web 管理界面。

## 功能

- 📊 仪表盘 - 系统监控
- 💬 聊天 - 实时对话
- 🧠 记忆 - 记忆管理
- ⏰ 任务 - 定时任务
- 🤖 模型 - 模型管理
- 🛠️ 技能 - 技能浏览
- 💬 会话 - 历史会话
- 📊 统计 - 使用统计
- 📋 日志 - 日志查看
- 🌐 网关 - 连接状态
- 💻 终端 - Web 终端
- ⚙️ 设置 - 系统配置

## 部署

```bash
chmod +x deploy.sh
./deploy.sh
```
